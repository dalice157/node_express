# node_express
## 使用教學
- 鍵入 express -e 資料夾名稱
他會自己新增一個資料夾，裡面就會有基本的資料，結構如下：
folder name
├── app.js
├── bin
│   └── www
├── package.json
├── public
│   ├── images
│   ├── javascripts
│   └── stylesheets
│       └── style.css
├── routes
│   ├── index.js
│   └── users.js
└── views
    ├── error.ejs
    └── index.ejs
- 鍵入 cd 資料夾名稱 && npm install
